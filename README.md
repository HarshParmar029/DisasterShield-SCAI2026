# DisasterShield - SCAI 2026

**AI-Powered Multi-Hazard Early Warning System with Edge Intelligence & Remote Sensing**

**Project for SCAI 2026**  
**Track:** Smart Sensing & Intelligent Electronic Systems (T1)  
**Institution:** Marwadi University

## Problem Statement
Climate change is increasing natural disasters (floods, wildfires, droughts) causing huge losses. Existing systems are expensive and not real-time/localized.

## Solution
Low-cost Raspberry Pi based edge computing system that:
- Detects hazards using multi-sensors + camera
- Uses AI (OpenCV + TensorFlow Lite) for real-time detection
- Fuses sensor data with simulated remote sensing
- Sends timely alerts via IoT dashboard

## Features
- Real-time smoke/fire detection using CNN
- Flood & drought risk prediction
- Sensor fusion for high accuracy
- Offline capable edge AI
- Web dashboard + SMS alerts

## Tech Stack
- **Hardware:** Raspberry Pi 4/5, Pi Camera, DHT22, Soil Moisture, Rain Sensor, MQ-135
- **Software:** Python, OpenCV, TensorFlow Lite, Flask/ThingSpeak
- **Datasets:** Kaggle Wildfire Dataset, PlantVillage

## Current Progress (as of July 2026)
- Hardware setup & sensor integration completed
- OpenCV-based vision system working
- Basic ML model training started
- IoT dashboard in progress
